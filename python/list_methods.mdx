# Module: Lists and Dictionaries

## Lesson: List Methods (append, pop, sort, etc.)

In this lesson, you will learn about some of the methods available for manipulating lists in Python.

## Adding Elements to a List

### append()

The `append()` method adds an element to the end of a list. Here is an example:

```python
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)
```

Output:
```
[1, 2, 3, 4]
```

### insert()

The `insert()` method adds an element at a specific index in a list. Here is an example:

```python
my_list = [1, 2, 3]
my_list.insert(1, 4)
print(my_list)
```

Output:
```
[1, 4, 2, 3]
```

## Removing Elements from a List

### pop()

The `pop()` method removes and returns the element at a specific index in a list. If no index is specified, it removes and returns the last element in the list. Here is an example:

```python
my_list = [1, 2, 3]
removed_element = my_list.pop(1)
print(my_list)
print(removed_element)
```

Output:
```
[1, 3]
2
```

### remove()

The `remove()` method removes the first occurrence of a specified element from a list. Here is an example:

```python
my_list = [1, 2, 3, 2]
my_list.remove(2)
print(my_list)
```

Output:
```
[1, 3, 2]
```

## Sorting a List

### sort()

The `sort()` method sorts the elements of a list in ascending order. Here is an example:

```python
my_list = [3, 1, 2]
my_list.sort()
print(my_list)
```

Output:
```
[1, 2, 3]
```

You can also sort the list in descending order by passing the `reverse=True` argument to the `sort()` method:

```python
my_list = [3, 1, 2]
my_list.sort(reverse=True)
print(my_list)
```

Output:
```
[3, 2, 1]
```

## Other List Methods

### len()

The `len()` function returns the number of elements in a list. Here is an example:

```python
my_list = [1, 2, 3]
print(len(my_list))
```

Output:
```
3
```

### index()

The `index()` method returns the index of the first occurrence of a specified element in a list. Here is an example:

```python
my_list = [1, 2, 3, 2]
print(my_list.index(2))
```

Output:
```
1
```

### count()

The `count()` method returns the number of occurrences of a specified element in a list. Here is an example:

```python
my_list = [1, 2, 3, 2]
print(my_list.count(2))
```

Output:
```
2
```

## Conclusion

In this lesson, you learned about some of the methods available for manipulating lists in Python. You learned how to add and remove elements from a list, sort a list, and get information about a list. With these skills, you can use lists to store and manipulate collections of values in your Python programs.
